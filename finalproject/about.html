             This User Application is a guide for users to use the System Application for Citizenship of Setiu District 
             Office. Detailed description of the steps and means of using the system regularly are given to avoid problems 
             in the implementation and use of the system by the user. Setiu District Office Citizenship Application System 
             is a system to apply for citizenship status as a people of Terengganu for the residents in Setiu District. The 
             system can be accessed online and applicants can fill out forms application even if it is located. The system 
             aims to make it easier for people to who wish to apply for citizenship status without having to go to the office.
